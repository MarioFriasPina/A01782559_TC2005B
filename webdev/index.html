HTML Cheat Sheet

a- Hyperlinks

Sirven para hacer un link de una página a otro recurso en la web, sea otro archivo en el servidor web, u otra página web.

Por ejemplo: El siguiente link muestra un video de cómo funcionan los hyperlinks!

p- Parrafos

Sirven para escribir parrafos dentro de una pagina web

Por ejemplo:

Este es un parrafo

h1, h2, ..., h6- Encabezados

Sirven para escribir encabezados de diferentes tamaños, siendo h1 el mas grande y h6 el mas pequeño

Por ejemplo:

Este es un encabezado de tamaño 1

Este es un encabezado de tamaño 6

div- Elemento de bloque

Sirven para dividir secciones de un documento

Por ejemplo:

Estoy dentro de un div

span- Elemento de linea

Sirven para seleccionar pequeñas porciones de texto, imagenes, etc

Por ejemplo:

Estoy dentro de un span

strong- Texto de Importancia

Sirve para indicar que un texto tiene gran importancia, usualmente en negritas

Por ejemplo:

Este es un texto importante

i- Letras Italicas

Sirve para cambiar el formato de un texto a italicas

Por ejemplo:

Este es un texto en italicas
