HTML Cheat Sheet

a- Hyperlinks

Sirven para hacer un link de una página a otro recurso en la web, sea otro archivo en el servidor web, u otra página web.

Por ejemplo: El siguiente link muestra un video de cómo funcionan los hyperlinks!

p- Parrafos

Sirven para escribir parrafos dentro de una pagina web

Por ejemplo:

Este es un parrafo

h1, h2, ..., h6- Encabezados

Sirven para escribir encabezados de diferentes tamaños, siendo h1 el mas grande y h6 el mas pequeño

Por ejemplo:

Este es un encabezado de tamaño 1

Este es un encabezado de tamaño 6

div- Elemento de bloque

Sirven para dividir secciones de un documento

Por ejemplo:

Estoy dentro de un div

span- Elemento de linea

Sirven para seleccionar pequeñas porciones de texto, imagenes, etc

Por ejemplo:

Estoy dentro de un span

strong- Texto de Importancia

Sirve para indicar que un texto tiene gran importancia, usualmente en negritas

Por ejemplo:

Este es un texto importante

i- Letras Italicas

Sirve para cambiar el formato de un texto a italicas

Por ejemplo:

Este es un texto en italicas

u- Letras Subrayadas

Sirve para cambiar el formato de un texto subrayado

Por ejemplo:

Este es un texto subrayado

u- Letras Negritas

Sirve para cambiar el formato de un texto en negritas

Por ejemplo:

Este es un texto en negritas

strike- Letras rayadas

Sirve para cambiar el formato de un texto rayado, es depreciado en HTML5, se usa s o del

Por ejemplo:

Este es un texto cancelado

small- Letras pequeñas

Sirve para cambiar el formato de un texto pequño

Por ejemplo:

Este es un texto pequeño

em- Texto Enfatizado

Sirve para indicar que un texto tiene enfasis, usualmente en italicas

Por ejemplo:

Este es un texto importante

header- Contenedor para contenido introductorio

Sirve para contener contenido introductorio o links navegacionales

Por ejemplo:

Estoy en un header

Yo tambien

section- Contenedor de secciones

Sirve para contener y definir secciones en un documento

Por ejemplo:

Estoy en la seccion 1

1!

Estoy en la seccion 2

2!

article- Contenedor de contenido independiente

Sirve para especificar contenido independiente, no cambia el contenido automaticamente

Por ejemplo:

Estoy en el articulo 1

1!

Estoy en el articulo 2

2!

aside- Contenedor de contenido aparte

Sirve para especificar contenido aparte del que lo rodea, no cambia el contenido automaticamente

Por ejemplo:

Este contenido es principal

Este es aparte

footer- Contenedor para contenido de pie de pagina

Sirve para contener informacion de autor, copyright, contacto, navegacion, etc

Por ejemplo:

Estoy en un footer

Yo tambien

nav- Contenedor para links

Sirve para definir un conjunto de links de navegacion

Por ejemplo:

Link 1Link 2Link 3

Sirve para insertar una linea horizontal que separa bloques en una pagina web

Por ejemplo: 1 2 3 ...

ul- Lista no ordenada

Sirve para crear una lista no ordenada

Por ejemplo:

  • Elemento 1
  • Elemento 2

ol- Lista ordenada

Sirve para crear una lista ordenada numericamente

Por ejemplo:

  • Leche
  • Chocolate
